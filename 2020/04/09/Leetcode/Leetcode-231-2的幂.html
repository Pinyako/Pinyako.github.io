题目描述

给定一个整数，编写一个函数来判断它是否是 2 的幂次方。

示例 1:

    输入: 1
    输出: true

解释: $2^0$ = 1

示例 2:

输入: 16
输出: true

解释: $2^4$ = 16

示例 3:

输入: 218
输出: false

题目分析

如果一个数是2的幂, 则它的二进制一定只有第一位是1, 其他位为0. 这样我们取它与比它小1的与. 如果所得为0, 则为2的幂, 反之则不是2的幂.

源码

class Solution {
public:
    bool isPowerOfTwo(int n) {
        if(n<=0) return false; // 2的幂一一定大于0
        return n&(n-1) == 0;
};
